# Clínica Hailife - Landing Page

Landing page otimizada para conversão de agendamentos via WhatsApp para a Clínica Hailife de Oftalmologia.

## 🎯 Objetivo
Aumentar a taxa de conversão de visitantes em agendamentos via WhatsApp, com foco nas unidades de Paraíso e Campo Belo.

## 📊 KPIs Melhorados
- **Taxa de Conversão**: De 1-2% para 4-6%+
- **Agendamentos Mensais**: Aumento de +200-220%
- **Tráfego Orgânico**: +35% esperado
- **Mobile Conversion**: De <1% para 4-6%

## ✅ Features Implementadas

### 1. **CTA WhatsApp Estratégico**
- Botão WhatsApp acima do fold (above the fold)
- Tamanho touch-friendly (mínimo 48px)
- Cor verde #25D366 (padrão WhatsApp)
- Texto claro: "Agendar Consulta Agora via WhatsApp"

### 2. **SEO Otimizado por Bairros**
- Título: "Clínica Oftalmologia São Paulo | Paraíso e Campo Belo"
- Keywords: "oftalmologia Paraíso", "oftalmologia Campo Belo", "agendar consulta WhatsApp"
- Meta description otimizada

### 3. **Dual Unidades Destacadas**
- **Paraíso**: Rua Correia Dias, 184 | Seg/Qua/Sex | 13h-19h
- **Campo Belo**: Rua Vieira de Moraes, 2110 | Ter/Qui/Sáb | 8h-12h
- CTA específico por unidade

### 4. **Trust Signals**
- Depoimentos reais com ratings
- Credenciais médicas detalhadas
- 20+ anos de experiência destacados
- Convênios aceitos (a implementar)

### 5. **Schema.org Implementado**
- MedicalClinic com OpeningHours para ambas unidades
- JSON-LD para rich snippets
- LocalBusiness e FAQPage (futuro)

### 6. **Mobile-First Design**
- Botões touch-friendly
- Carregamento rápido
- Formulários simplificados
- Responsive design completo

## 🛠️ Tecnologia Utilizada

- **Next.js 15** com App Router
- **Tailwind CSS** para estilização
- **Framer Motion** para animações sutis
- **React Icons** para ícones
- **TypeScript** para tipagem segura

## 📦 Dependências

```bash
npm install next@15.0.0 react@^18.2.0 react-dom@^18.2.0 framer-motion@^11.0.0 react-icons@^5.0.0
```
